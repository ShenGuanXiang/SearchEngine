    暑假来【乌镇】怎么玩？白天到晚上包你嗨不停！

    • 出发日期/2021/07/18

    • 天数/1天

    乌镇

    什么是乌镇

    「来过，未曾离开！」这是乌镇的广告语！

    去过一次乌镇，就会在脑海里留下对乌镇的各种画面！这便是乌镇的魅力。

    乌镇是桐乡一座历史悠久的江南古镇，主要景区分为西栅和东栅，赏西栅夜景、坐摇橹船、看民俗表演，是游人不能错过的体验。乌镇也是黄磊和刘若英主演的《似水年华》的取景地。黄磊还在这儿开也一个酒吧，兴许，偶尔能偶遇一下。

    东栅和西栅相距约1.5公里，只呆一天的话，可以上午早点到乌镇，先去东栅游览两三小时，午饭后就可以一直呆在西栅游览了，一直玩到晚上在西栅看夜景，以及一些娱乐表演。

    而乌镇，现在也不仅仅是一个古镇，每次去乌镇都会看到不同的玩法！

    另外，乌镇景区的民宿、酒店、店铺等等都是统一管理的，到达游客中心的时候就需要办理入住手续，景区里面的商品价格也是统一的，所以不存在入口处便宜点，中心位置贵得离谱这样的情景。

    乌镇是早上九点才可以入内的，所以如果是住在里面的游客，如果不想吃酒店里的早餐，可以选择早茶客（早茶客和酒店早餐二选一）。这时候外面的游客还没有进来，所以人不多，这样的情境下，非常舒服。

    乌镇无尽夏生活节

    乌镇如意桥夜市

    乌镇“无尽夏”生活节闪亮登场，活动时间为2021年7月11日——8月27日。

    乌镇以“‘暑’乐乌镇，嗨玩一夏”为主题，给游客朋友们打造了包括如意桥夜市、戏剧游乐场、儿童剧嘉年华三大板块，玩一天就可以将乌镇、夏日、好玩、文化沉浸、艺术体验等一网打尽。

    吃过晚饭后就去了夜市，夜市在乌镇标志地-白莲塔边上，与白莲塔隔岸相望。

    天还没黑的时候，夜市的灯光就开始亮起来了，游客们也陆续到达，这儿有各种风味美食、夜宵、以及一些文创艺术，还有几个非遗传承，比如人像剪纸，简直比照片还惟妙惟肖。

    逛完夜市后，还可以来乌镇的水剧场，这儿有超大的观众席，可容纳千百余人，露天的表演可以让你在星空下放松心情。

    这儿的表演大致有乐队驻唱、小品表演、歌舞表演等等。偶有节目还会来观众席与观众一起互动，可不要抗拒这个夏日独有的相遇哦！

    评论
    早茶客

    每天早上，这儿会有人摆摊，也有摇着船来的，停在岸边，贩卖一些当地的瓜果蔬菜，游客也可以参与其中，挑选自己喜欢的物品，收入囊中。

    水上集市，也是一条早餐街，买完东西后，坐下来吃个早餐，喝点早茶，馄饨、汤圆、油条、豆腐脑、豆浆、米糕……超多吃食，生活气息非常浓厚，人来人往中氤氲出一个繁盛江南。

    不过，这些都需要入住在乌镇才能体会到，因为乌镇是早上九点才可以进入景区。景区特意留给住在里面的游客一个清静的清晨。

    可以让你看看清晨没有人的乌镇，体验到最原始的生活状态，在没有人游客打扰的情况下，放松下来，逛逛水市，吃吃早餐。

    评论
    水上婚礼

    乌镇是一个有1300年建镇史的江南古镇，自古是水乡泽国，镇内河网纵横交错，“十”字型的主河道将乌镇自然划成四条大街，桥成了贯通东西南北的要塞，而桥下的船只，千余年来，都是重要的交通工具。包括婚娶，也会在水路上进行！

    乌镇就复原了以前用船来迎娶新娘的习俗，偶遇一场水乡婚礼秀。在安渡坊码头-灵水居-枕西码头间穿梭，可以说是非常热闹。

    乌镇木心美术馆坐落在木心先生的故乡乌镇，毗邻西栅的乌镇大剧院。这座个人美术馆首次完整展示木心先生的绘画与文学作品，并由木心的学生、画家陈丹青先生出任馆长。

    在馆内共有5间永久性专馆，长期陈列木心作品。展馆包括绘画馆、文学馆、狱中手稿馆、影像厅等，并有两座特展馆，能看到包括“狱中手稿”在内的木心各个时期的画作。

    美术馆有一个阶梯式的图书馆，观众可以在木质阶梯上小坐，欣赏美术馆后院的景色，还能阅读木心《文学回忆录》等书籍。

    边上还有一处旋转楼梯，拍照打卡一定要来这儿，真的是随便怎么拍都非常漂亮。时间不够，就很快速的拍了几张照片，回来看，毫毛构图，期待下次再去拍了！

    锦堂行馆是乌镇行馆之一，坐落于西栅景区的老街中段，闹中取静，南临石板老街，东接洛神SPA，北可隔河眺望天然湿地，西有清流直通市河，地理位置优越。

    锦堂行馆由乌镇一造船世家的私宅改建而得，为两层古典硬山式园林建筑，非常典型的明清建筑风格。

    酒店早餐也是非常丰富的自选早餐，中式西式都有。

    怎么去乌镇

    交通攻略

    1、乌镇没有机场，如坐飞机，可到邻近桐乡的上海、杭州等城市，再转火车或长途大巴。

    2、火车可至桐乡火车站，再做公交K282到乌镇汽车站。

    3、长途汽车直接坐至桐乡汽车站再转乌镇汽车站，上海、杭州、西塘等都有直接到达乌镇汽车站的班次。 以上方式到达乌镇汽车站后，可坐公交K350，到乌镇东栅、西栅景区；东栅西栅景区之间有免费接驳车可以乘坐，也可以步行。

    乌镇汽车站——桐乡火车站（K282）

    桐乡火车站——乌镇汽车站：每天有39班 首班7:25，末班21:40

    乌镇汽车站——桐乡火车站：每天有38班 首班6:10，末班20:35

    注：夜班车，始发站为乌镇西栅景区（西栅出发班次为：18:15、18:50、19:50、20:35）

    乌镇汽车站——桐乡客运中心（K231）

    桐乡客运中心——乌镇汽车站：每天有46班 首班6:00，末班18:00

    乌镇汽车站——桐乡客运中心：每天有46班 首班6:00，末班18:00

    注：夜班车，始发站为乌镇凤仙新村（乌镇凤仙新村出发班次为：18:30、19:20、20:10、21:00）

    自驾游

    1、上海-乌镇：全程仅80分钟的车程，走申嘉湖高速公路到乌镇，乌镇出口下，10分钟可达乌镇。

    2、苏州到乌镇：全程仅6 0 分钟的车程，走乍嘉苏高速公路，到嘉兴转申嘉湖高速，到乌镇出口下高速，10分钟可达乌镇。

    3、南京-乌镇：

    （1）从南京到乌镇3 0 0 公里左右，从南京出发，走杭宁高速湖州北出口下到一级（盐湖）公路直接到乌镇。

    （2）走沪宁高速到苏州转乍嘉苏高速公路，到嘉兴，转申嘉湖高速，到乌镇出口下高速，1 0 分钟可达乌镇。

    4、桐乡-乌镇：从桐乡到乌镇1 3 公里，从桐乡出发，走一级（盐湖）公路直接到乌镇。

    5、杭州-乌镇：　全程仅5 0分钟的车程。

    （1）走申嘉湖杭高速，（沿S1 3 行驶49.9公里，朝嘉兴/上海方向，稍向右转），转到申嘉湖高速( 沿S12行驶4.3公里，到达乌镇出口)，乌镇出口下高速，向北一直开( 沿姚太线行驶2 . 7 公里) ，就到乌镇了。

    （2）走沪杭高速公路到高桥出口下，经桐乡，走2 0公里一级公路，2 0 分钟可达乌镇。（或从杭州走,320国道一直到桐乡）。

    6、嘉兴-乌镇：从嘉兴到乌镇4 0 公里左右，从嘉兴出发，走3 2 0 国道到桐乡，再经桐乡到达乌镇；或者直接嘉兴到乌镇。

    7、宁波-乌镇：全程约2-3小时的车程。

    （1）跨海大桥高速，过桥，（海盐枢纽，又称西塘桥枢纽）转到乍嘉苏高速，（观音桥枢纽），转到申嘉湖高速，乌镇出口下高速，向北一直开，就到乌镇了。

    （2）走杭甬高速公路，到杭州转沪杭高速，到屠甸下高速，经桐乡，走2 0 公里一级公路，20分钟可达。

    8、温州-乌镇：从温州上高速，沿宁波方向上宁波绕城高速，上杭州湾大桥，下桥后走乍嘉苏高速，转申嘉湖高速，乌镇出口下，5 个多小时车程。

    9、无锡-乌镇：太湖大道——沪宁高速——苏嘉杭向南——乍嘉苏高速向南——申嘉湖高速——乌镇，车程为135公里。
