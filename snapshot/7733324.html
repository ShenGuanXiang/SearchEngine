    【平潭留白】追不上的蓝眼泪，追得上的海风。

    • 出发日期/2021/04/03

    • 天数/3天

    • 人均费用/1500元

    • 人物/家庭

    赴一场蓝色之约。

    若不是因为疫情的关系，往年的这个时候，

    平潭岛应该已经迎来了一年中人气最旺的“高光时刻”。

    进入4月，平潭岛便进入了“蓝眼泪”的爆发期，

    浩浩荡荡的“追泪大军”纷至沓来，只为一睹荧光之海的风采。

    所谓蓝眼泪，其实是一种海中的浮游生物，

    在受到海浪拍打等刺激时，便会发出蓝色的荧光，

    随着海浪层层推向岸边，仿佛是在海中起舞的萤火虫，

    就如同“蓝眼泪”这个诗意的名字一样，

    神秘，又浪漫。

    想要在平潭岛观赏到蓝眼泪，其实并不难，

    然而我却依然遗憾地与它擦肩而过。

    尽管这场因蓝眼泪而兴起的旅行，最终没能得偿所愿，

    但经历了被疫情反复打乱出行计划、

    在江南连绵不绝的春雨里宅到发霉的小确丧之后，

    只是望着海面发呆，享受一下吹吹海风的自在，

    就已经足够治愈了。

    你问我要去向何方，

    我指着大海的方向。

    评论

    海 缘 / 猴 研 岛 。

    ▲“68海里风景区”是猴研岛的另一个名字

    猴研岛其实并没有猴，别说猴了，甚至连像样的树也没有几棵，

    只有一大片临海的、姿态嶙峋的怪石头，

    或浅黄，或深灰，或赭红，与看起来平平无奇的沙滩，和蔚蓝色的大海， 

    组成了我的平潭初印象。 

    虽然没有猴，倒是留下了一个与猴相关的传说。

    有一只母猴带着一只小猴前往台湾岛，不料途中遇到风浪，

    母猴自己漂回了猴研山，而小猴则漂到了新竹南寮渔港。 

    之后的故事走向就变得似曾相识了， 母子俩就这样一直隔海相望，渐渐就变成了化石。 

    虽是传说，岛上似乎也并没有一块猴子形状的石头， 

    但当你知道了这里其实是中国大陆距离台湾本岛最近的地方时， 

    这个故事读起来，就别有一番风味了。 

    68海里，也就是126公里， 

    这便是猴研岛距离小猴子的所在地——新竹南寮渔港的距离， 

    而岛上刻有“68海里 中国·平潭”朱红色大字的标志性石碑， 

    自然成为了游客们争相打卡留念的“背景墙”。 

    整片景区目前是免费开放的状态，还有很多区域正在开发之中，

    期待它全部完工的那一天，也期待福建到宝岛的高铁正式通车的那一天。 

    那个时候，我们之间的距离， 不再是68海里。

    ▲海的另一边，又是谁的故乡

    ▲猴研岛一角

    ▲石碑打卡正确

    评论

    海 风 / 大 福 湾 。

    大福大福，不是草莓大福，而是海味的大福。

    一个听起来就感觉福气满满的名字，风景也着实让人大饱眼福。

    海边的一块礁石上，写着一个巨大的“福”字，

    远处还有一座颇为上镜的白色情人桥。

    一对幸福的新人，正站在桥上拍摄婚纱照，

    新娘依偎在爱人身旁，头上的白纱随风飘舞，

    大海啊，果然是浪漫的催化剂，让人不经意间，便牵起了嘴角。

    这里几乎是平潭岛的最南端了，游客寥寥，海边的餐厅冷冷清清，

    这片集住宿、垂钓、环岛游于一体的休闲海湾，不知何时才能恢复往日的热闹。

    但或许，就这样安安静静的，也挺好。

    ▲想要抵达白色情人桥，先要登上这艘“大福号”

    ▲是不是有一种“小希腊”的feel偷笑小蚂

    ▲近观不如远望

    大福湾的一旁便是将军山，不过我去的时候实在太晒了，

    便只是远远地看了一下。

    山越往上走越荒凉，如果你时间充裕，

    可以爬到山顶的石碑下看一看，也许会有意外收获呢。

    评论

    海 趣 / 坛 南 湾 。

    坛南湾给我的感觉，是大而自在。

    这里大概是我此行遇到游客最多的景区了，

    享受这片海的方式，亦有很多种。

    你可以脱下鞋子，踩在柔软的沙滩上，

    或是躺在树荫下的吊床上，枕着海风，打个小盹；

    你可以沿着长长的漫步道散心，

    体力好的话，还可以拾级而上，登高望远；

    你可以坐帆船，租帐篷，捡贝壳，

    把整整一个下午的时间，留在这里。

    很多人都是全家出动，

    孩子们纷纷拿出自己的小铲子、小水桶，

    创造着千姿百态的“沙雕艺术品”，

    大人们则围坐在一旁，热聊着各自的话题。

    这里也是岛上最容易看到蓝眼泪的区域，

    一批透明的海景房，正对着眼前无垠的大海，

    或许不久以后，这里将成为平潭岛新的网红景点。

    ▲只是静静地望着，就已足够美好

    ▲去年4月，这些海边的玻璃房子已经差不多建好了，现在不知道有没有正式营业

    ▲走进了一座已经盖好的房间，捕捉到了从玻璃房内望向大海的第一视角

    评论

    海 梦 / 北 港 村 。

    除了蓝眼泪，北港文创村也是名声在外，

    这都要得益于当年红遍大江南北的现象级综艺——《爸爸去哪儿》。

    这座原本海边安静的小渔村，如今已经完全商业化了，

    到处都是民宿、餐厅和工艺品小店，游客们穿梭其中，找寻着自己心中的“宝藏”。

    北港村的特色是石头房子，也就是以花岗岩为主要材料，搭建的古石厝老屋，

    青灰色的房子连成一片，沿山势而上，看上去有点压抑。

    不过，在充满平潭风情的石头民宿屋里住上一晚，

    对于许多外地游客来说，依然是充满吸引力的行程。

    加上这里弥漫的文艺气息，这座点“石”成“金”的小渔村，也有点平潭“曾厝垵”的味道了。

    ▲北港村一瞥

    ▲山上雾气缭绕

    ▲北港村的石头屋厝

    ▲没看到蓝眼泪，看到了蓝色的小螃蟹大笑小蜂

    在北港村里，有一家人气很旺的小店——石头会唱歌。

    这里是北港村最早入驻的民宿之一，老板来自台湾，还兼营着一家售卖纪念品的咖啡屋，

    门口的长桌上摆放着一排大小不一、形状各异的石头，每一块都对应着一个音符，

    游客们纷纷拿着小石锤敲击，演奏出一段段磕磕绊绊的旋律。

    所以，平潭的石头，是真的会唱歌哦偷笑小蚂

    ▲你的乐器已就位

    ▲这里也可以寄明信片

    ▲走累了，就坐下喝杯咖啡吧咖啡

    不知熙熙攘攘的游客，

    会不会惊扰了这座小渔村的美梦呢？

    评论

    海 浪 / 仙 人 井 。

    其实这个景区真正的名字叫“东海仙境”，不过这个title属实有点夸张了，

    但是位于此的“仙人井”，还是非常值得一看的。

    这座直径近50米、深40多米的天然海蚀竖井，仿佛被一把斧子从中间劈开，

    在两块巨大的断崖之间，海浪翻涌着拍打岸上的圆石，

    风越大，浪越高，海浪的声音，也越来越气势磅礴。

    圆石踩着很容易打滑，一个大浪就能把你脚下所站立的位置瞬间吞没，靠近拍照需小心谨慎。

    ▲悬崖间的仙人井

    ▲神奇的大自然

    看完了仙人井，还可以沿着台阶，登上旁边的观景台，远眺大海。

    可惜我去的那天是阴天，大海一片灰蒙蒙，不过也多了几分苍茫之味。

    ▲面朝大海的观景台

    ▲浪层层翻卷，拍打着那些经年不语的礁石，奏出澎湃的交响

    评论

    海 境 / 生 态 廊 道 。

    这是我平潭之行最喜欢的一段旅程，所以直接拿来做了这篇游记的头图和封面，

    私以为来过这里，就没有必要再去长江澳风力田景观区了。

    虽然叫“生态廊道”，其实并没有专门的步行和骑行道，大家都是自驾来的，

    只有我，完全靠步行，走完了五个观景台，也是佩服自己的体能和耐性。

    因为一路都有大海、风车相伴，这两个多小时的徒步，倒也不觉得无趣和辛苦，

    每次都期待着下一个观景台，会是怎样的蓝色画境。

    最特别的是2号观景台，有一段长45米的悬崖玻璃栈道，是最热门的游客打卡地。

    踩在透明的玻璃上，脚下就是大海，这样神奇的体验，确实不容错过。

    不过这里风非常非常大，穿裙子的集美，小心走光哦坏笑小蜂

    ▲我的徒步之旅，从看到这座大风车开始

    ▲走了一段，身侧就豁然开朗

    ▲停泊的渔船、碧蓝的大海和远处的风车，组成了平潭最美的明信片

    ▲风车是你一路上的最佳伴侣

    另一处5号观景台的视野也是一级棒，一面可以看到大海，一面看到村落和整个长江澳的样子，

    让因为时间关系，而放弃了原本计划中的长江澳的我，也不觉得遗憾了。

    ▲除了没有遇上晴天，这段徒步之旅还是非常nice的

    评论

    海 街 / 海 坛 古 城 。

    国内的首座海岛旅游古城，但其实就是景区标配的那种“小吃街”既视感，

    只不过从一条街道，变成了一座迷你的街区，涵盖了各色餐厅、闽台风味小吃和特色旅游商品。

    景区内也会有一些民俗表演，但我们去的时候没什么游客也没有表演，显得非常冷清。

    这里位于从坛南湾回到市区的路线上，如果你正好路过，可以来转转，

    专程赶来，就大可不必了。

    ▲我去的时候没有开放登楼

    评论

    海 味 / 大 海 小 鲜 。

    靠山吃山，靠海吃海，

    地域的密码，都藏在这八方风味之中了。

    来到平潭，自然鲜味当道鱼

    餐厅平潭县 / 破店蹄髈星星星星星星星星

    算是岛上评价最好的店了，有多家连锁。

    招牌蹄髈每桌必点，肉和笋干都很入味，好吃！

    佛跳墙算是闽南菜的扛把子了，配料爆炸多，鲍鱼、干贝、海参、猪肚、猪蹄、鸡胗、牛筋、鹌鹑蛋……不过汤喝起来有一股奇怪的味道，不是想象之中的鲜甜，不是我的style。

    老豆腐过于清淡了，汤好喝，但豆腐没有什么味道，油淋菜心倒是意外的惊喜，菜炒得有点生，但又脆又甜。

    鱼丸汤虽然只有四颗鱼丸，但福建的鱼丸和我们这里的完全不是一回事，个头超级大，非常Q弹。

    炒米线料足量大，实在吃不完，打包回酒店当夜宵了。

    ▲拿菜名当店名，就是有底气

    ▲一盘不普通的菜心

    ▲单点一份做夜宵也不错

    ▲XL号的鱼丸

    ▲这佛跳墙看起来就很补了

    餐厅平潭县 / 姑奶奶炒饭星星星星星星星星

    路边随便找的一家小炒店，味道超乎意料的好。

    鱼头诱惑香辣开胃，小份已经够三个人分享了；紫菜筒骨汤鲜到起飞，紫菜不愧是平潭的特产，确实好吃；小炒肉也是亲切的家常味道。

    这家有免费的饮料和水果，特别是米浆，很！好！喝！

    ▲吸饱汤汁的豆腐和面条是灵魂

    ▲紫菜太鲜了

    餐厅北港村 / 海角茶餐厅星星星星星星

    随意找的一家店，口味普普通通。

    印象比较深的是杂鱼，各种鱼都很新鲜，而且非常入味；椒盐虾也不错，虾肉鲜甜。

    鱼丸汤分量惊人，虽说是鱼丸，但却不是圆的，而是长条形，非常有嚼劲，这也算是平潭的特色了。

    八爪鱼泥沙没有吐干净，味道一般。

    ▲神奇的条形鱼丸汤

    ▲杂鱼赞赞小蜂

    评论

    番 外 / 追 泪 tips 。

    虽然我并没有追泪成功，但也可以给各位提供一些不成熟的小建议。

    1.北部的北港村和南面的坛南湾都是比较容易看到蓝眼泪的区域，想要追泪建议住在这两个区域附近（北港村更佳，因为还可以体验当地的特色民宿），蓝眼泪爆发期时，前往北港村和坛南湾的环岛路会非常非常堵，住在县城开车过去会不太方便。

    2.4月就已经进入蓝眼泪的观测期了，但晚上的海边风还是很大的，建议要穿一件厚一些的外套。

    3.如果想和蓝眼泪进行更加亲密的接触，可以参加当地的一些蓝眼泪观测团，会有渔船带你出海，但蓝眼泪的出现时机很难确定，所以即使参团也有可能看不到蓝眼泪。

    4.可以关注“平潭气象”、“平潭蓝眼泪预报”这两个微博，会每天发布蓝眼泪出现的概率、出现的位置等信息。

    ▲这段时间每天都会更新蓝眼泪的信息

    5.平潭岛最方便的出行方式是自驾，去往县城外景区的公交班次比较少，但人多的话打网约车也是划算的。

    PS：其实行程的第二天是可以看到蓝眼泪的，但因为那天晚上风太大，外面非常冷，而且我住在县城，打车过去不太方便，就放弃了捂脸小蜂

    最后，如果你也是因为蓝眼泪而来到平潭岛，

    那就祝你追泪成功吧爱心小蜂
