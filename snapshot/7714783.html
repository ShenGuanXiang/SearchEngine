    • 出发日期/2021/11/10

    • 天数/5天

    • 人均费用/1500元

    • 人物/三五好友

    • 玩法/深度游 自驾 

    前言

    说说这次旅行

    潮州是中国优秀旅游城市、对外开放旅游城市 、国家历史文化名城，是一座拥有1600多年历史的文化古城，潮汕文化是岭南文化的重要组成部分。潮汕地区对于广东来说还是个“处女地”，属于潮汕文化，和广府文化还是有很多的区别，潮州历史悠久，文化底蕴深厚。除了美食诱人外，这里对传统的传承还是很重视，各种古村古迹保留的不少。

    评论
    行程

    游玩建议：

    潮州的美食小吃遍布大街小巷，风光美景、古村古镇、非遗项目分布在潮州周边，公共交通肯定没那么方便。

    原来打算是自驾去，佛山禅城出发，全程440公里，约7个小时。

    考虑到长时间驾车不安全，最后选用第二套方案，从广州坐高铁到潮州，广州南站 → 潮汕站，210元，3小时以内到达，网上提前租好了车，从高铁出站，联系上租车公司，有工作人员送车到站，验车办手续，非常的方便，5天租车连油费、高速，全程下来不到1000元。

    必带物品：泳衣、水泡、防晒油、防蚊水、帽子、雨伞、雨衣、药油等。

    这是猪猪安排的五天四夜行程，作为参考，不会很紧凑，各个元素都有，每个点的玩乐时间管够，可以根据自身行程调整。

    Day1：广州南站 → 广济桥 → 古城牌坊街 → 开元寺 → 圆蛋酒料 → 潮味坊 → 都是梦想家酒店

    Day2：潮州大吴泥塑博物馆 → 潮州刺锈文化研学游基地 → 明鸿庄园熊博士 → 家和小吃店 → 东山湖景区 → 伟记甜汤 → 兴香宝牛肉丸 → 瀛洲狮头鹅 → 都是梦想家酒店

    Day3：凤凰天池 → 潮州单枞茶博物馆 → 阿哑浮豆腐 → 凤凰革命烈士纪念公园 → 凤翔峡风景旅游区

    Day4：李工坑村 → 狮峰村 → 绿太阳生态旅游度假区 → 陈德将军生平展示馆 → 龙湖古寨 → 官塘四季牛肉火锅店 → 超派火锅城 → 维也纳酒店

    Day5：广济桥 → 古城牌坊街 → 广州南站

    到潮州，必到潮州古城，这里有广济桥、牌坊街，吃喝玩乐齐。第一站就到这里了，这边停车不太方便，在酒店安顿好了，步行过去。

    广济桥，又称湘子桥，在潮州城东门外。形态多种多样，外观平稳威严，有如大殿。广济桥是浮梁结合结构，由东西二段石梁桥和中间一段浮桥组合而成，梁桥由桥墩、石梁和桥亭三部分组成。广济桥全长518米，中间浮桥长97.3米，中段浮桥由十八只木船横向并排连接而成，矶头墩各有一段石阶梯供行人上下浮桥。浮桥可启可闭，既可为行人过江，又可打开供船只通行、快速排洪。由于通航的需要，在下午5:30左右，浮桥会收起来，以便货船从江面通过。又形成一道“过河拆桥”的独特景色。

    猪猪这天16:40分到达，已经进不去，木八梭船也已拆开，后来一看，原来是执行冬季时间。走进广济桥是要购票的，普通票20元/人，优惠票10元/人。

    冬季参观时间周一~周五10:00-16:30，周末节假日：09:00-17:30，晚上19:00亮灯，去牌坊街逛逛吃吃，买点手信，够了时间去看夜景更是美得令人赞叹~由于节约用电，21:00左右灯光也关闭，去参观，记得控制好时间哦。广济桥附近停车位较紧张，步行或者骑个小电动更佳。

    在猪猪去过的地方，牌坊也不少见，但全国独一无二的文化景观，位于古城中心的潮州牌坊街，绝对是惊艳。潮州牌坊街位于潮州古城中心，是集非物质文化遗产、传统工艺、特色小吃等为一体的文化古街。

    23座中式明清石牌坊与两侧南洋风味的骑楼互为映衬，牌坊街店铺都是围绕潮州非遗：潮绣、手拉壶、陶瓷、工夫茶等，还有潮式美食：潮州三宝、牛肉火锅、粿条、甘草水果…吃喝玩乐尽在此处，逛完还能去旁边的广济桥看看。

    到广济桥，都会到牌坊街，这基本都是一体的，第一次来潮州游玩的，都是必到的打卡点，古老文化和现代商业相融合，商业氛围非常的浓，不是看到那些牌坊，不会想到这里的“古”。这里店铺几乎下午才开业，食物卖完就收摊，吃的喝的，民宿客栈也有，尤其多的是卖手信的，都是卖的“潮州三宝”，每家都标榜老字号，自己多长个心眼哦，价格方面还好，不会漫天开价的，还是很实在。牌坊街的夜景会给白天更好看，买了一条特色“广济桥”文旅雪糕，15元，同行的小姐姐说要打个卡。

    印象中的宝刹名寺，都是在高山茂林之中，有种神秘之感，不过有“粤东地区第一古刹”之称，潮州的开元寺却屹立于闹市里。开元寺前身为荔峰寺，在唐代开元二十六年敕建开元寺，元代时改为“开元万寿禅寺”，明代再改为“开元镇国禅寺”，俗称开元寺一直用到现在。开元寺一直是历朝祝福君主、宣讲官府律令的场所。寺里地方宽敞、殿阁壮观、圣像庄严、文物众多、香火鼎盛。

    整座寺院保留了唐代平面布局，兼俱了宋、元、明、清各个不同朝代的建筑艺术，香火鼎盛。 70年代，开元寺曾遭严重破坏，后来进行全面维修。

    开元寺现在也在牌坊街、广济桥、潮州古城这一带旅游景点之内，到这边参观游览，也可顺道去添个香油。开元寺是免费参观的，刷身份证进寺参观，需出示健康码、测体温。进寺入口有提供免费的香，不用特意在门口卖香的购买。参观时间，冬令时：6:30-17:00。

    开元寺就在牌坊街处，路过的时候想着先到牌坊街逛完再过来，就在门口咔了几张，不过后来逛街逛过了，回头的时候，关门了哦，错过了，以后有机会再去。

    社会主义新农村，山清水秀狮峰村

    狮峰村白鹭湖

    同行有曾经去过的朋友，一直推荐狮峰，说非常不错的村子，到达后看看，确实也是，尤其是航拍上，可以看到以前村子建造的布置与别不同，很壮观哦。

    潮安归湖镇著名的文化村和广东省文明村镇“狮峰村”，狮峰村的特色是山清水秀的自然景观，山光明媚，水色秀明，空气清新，绿色葱葱的自然风光，过千亩的大型湖泊，有围栏，有木栈桥，有荷花，还有高低错落有序的民宅，

    狮峰村自古以来重文兴教，明清时代，狮峰村就有“启明”、“智远”二个私塾，又称南校北校，1931年后，南北校合并为西丰小学。1986年侨胞李义丰、李丁祥等人赠建“盛林书院”，狮峰小学也建于此处，现在孔子像在书院中依然诲人不倦。村内分布着明清留下来的洋楼别墅，仿佛能看到当年文人在此讨论书文的景色。从高空往下看，更是美得不得了~

    狮峰村是革命老区村，也是广东省新时期精准扶贫重点帮扶村。更评为“广东十大最美乡村”，村里建设的白鹭湖风光不错，上面种满了荷花，荷花盛开的时候那是美不胜收。狮峰村现在绿树荫满径，白墙映青瓦，屋前屋后栽有小花，整洁的村道，粉刷一新的墙面，还有正能量的宣传图画和标语……一幅社会主义新农村的新画卷。

    尊奉自古至今的图腾文化，畲族起源地

    李工坑村
    3分/
    3篇游记中提到
    门票¥
    预订
    • 地址：广东省潮州市潮安区文祠镇
    • 查看详情

    没来过，还真不知道有这样一个少数民族，而且带有这么多的故事。

    潮安的凤凰山中隐藏着一条神秘的古村，尊奉自古至今的图腾文化，是畲族的起源地“李工坑村”，由李氏先居，称李公坑，1953年易名李工坑。这里主要姓雷，所以又称为雷乡。

    畲族与瑶族，黎族，苗族等少数民族类似，有自己民族风格的服饰、习俗、文化、语言等，畲族人世代居于凤凰山，凤凰就是族群的保护神。畲族吸收了潮州文化，有自己独特民族文化，建筑也带有潮州民居的痕迹。宗族祖祠改为畲族文化展览馆，里面还藏有描述了他们族群由来和发展衍生的经历和故事的“祖图”。作为一个吃货，也是当然要试试畲族独有的美食哦。主要是竹筒饭、烤鸡、香猪腊肉、竹笋干，还有炸蜂蛹和竺叶粿，竹筒饭由生到熟，现蒸现烤，闻着就香，馅料十足。蜂蛹肥肥的，口感酥脆。

    李工坑村保存较好的是语言、畲歌、传统节日。有语言无文字，村民用口述的形式叙述和继承历史，填补了无文字记载的不足。唱畲歌和跳畲舞是当地群众极为喜爱的文化活动。畲族有自己的宗教信仰，最重大且有代表性的民俗活动是联村祭祖习俗和“招兵节”活动。李工坑村以发掘畲族独特的民俗风情和神秘的历史传说，配合“山、水、林、果、茶”等自然资源，建成为畲族文化特色的旅游休闲观光点。

    免费参观的古村，商业氛围不算浓，傍晚时候拍照更有另一种魅力。

    喜欢逛古村古镇的，广东十大最美的古村落“龙湖古寨”是其中一个必打卡点。始建于1000年前南宋，龙湖古寨地处韩江中下游，傍江而筑，现存100多座古建筑，古寨整体布局结构十分讲究，按照九宫八卦而建，形成了“三街六巷”的工整格局，有“小潮州城”之称。古寨内府第祠堂、题词碑记、书斋学田林立，独特的人文气息和饱经沧桑的棕墙青瓦就是吸引人之处。

    龙湖古寨位置不错，在江边的大路边上，停车场有点远，停好车可以通过堤坝上的绿道慢慢走到古寨，古寨边上也有些免费停车位，先到先得。古寨寨门入口前，扩建了一个广场，休闲等候，或者团队集结很方便。古寨现在是免费参观，通过保存完好的寨门进去，一条算是毕直的青石板路，两端开满了各种小吃店，小丸子、蛋白肠、草粿、薄饼、潮州特产等应有尽有，不过下午才热闹，上午很多摊位还没开。重点参观的是巷子，走进巷子里，马上感觉像是穿越了时空，回到过去的峥嵘岁月，心里一片宁静安逸，拍景拍人超赞！不过游览过程中也发现古寨的保护措施不行，可能也和房子的产权有关，巷子里很多房屋倒塌或者残破，府第祠堂等都是紧闭大门，也进不了的，房子基本上没人居住，只有偶尔看到有些老人家在，村民都是搬到后面的新村里去居住了。
